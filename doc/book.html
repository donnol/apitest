获取图书信息

GET /apidoc/book

Request header:

  • Content-Type: application/json; charset=utf-8

Response header:

  • Content-Type: application/json; charset=utf-8

Param - Query

  • id (uint)

Return

  • code (int)
  • msg (string)
  • Book (object)

Error

  • {"code":1,"msg":"认证失败"}
  • {"code":2,"msg":"校验失败"}

Example:

Param 
id=1
Return 
{
    "code": 0,
    "msg": "",
    "Book": {
        "id": 1,
        "string": "jd",
        "page": 100
    }
}
